## ðŸš€ Getting Started

### Prerequisites

- **Python 3.7 or higher** installed on your system
- A text editor or IDE (VS Code recommended)
- Command line/Terminal access

### Installation

1. **Clone or download this repository:**
   ```bash
   git clone https://github.com/yourusername/python.git
   cd python
   ```

2. **Verify Python installation:**

## ðŸ’¡ Key Concepts Summary

### Day 01 - Fundamentals
| Concept | Description |
|---------|-------------|
| **print()** | Outputs text/data to console |
| **type()** | Determines the data type of a value |
| **Comments** | Text ignored by interpreter (documentation) |
| **Indentation** | Essential syntax element defining code blocks |

### Day 02 - Data Types
| Data Type | Example | Use Case |
|-----------|---------|----------|
| **int** | `10`, `-5` | Whole numbers, counting |
| **float** | `3.14`, `-2.5` | Decimal numbers, measurements |
| **str** | `"Hello"` | Text, names, messages |
| **bool** | `True`, `False` | Logical conditions |

### Day 03 - Variables and Operators
| Concept | Description |
|---------|-------------|
| **Variables** | Containers that store data values |
| **Naming Rules** | Must start with letter/underscore, case-sensitive |
| **Arithmetic** | `+`, `-`, `*`, `/`, `%`, `**`, `//` |
| **Comparison** | `==`, `!=`, `>`, `<`, `>=`, `<=` |
| **Logical** | `and`, `or`, `not` |
| **Assignment** | `=`, `+=`, `-=`, `*=`, `/=`, etc. |

### Day 04 - Control Flow and Logical Operators
| Concept | Description |
|---------|-------------|
| **if Statement** | Execute code based on a condition |
| **if-else** | Two alternative code paths |
| **if-elif-else** | Multiple conditions to check |
| **and Operator** | Both conditions must be True |
| **or Operator** | At least one condition must be True |
| **not Operator** | Reverses the result of a condition |

### Day 05 - Loops
| Concept | Description |
|---------|-------------|
| **for Loop** | Iterate over sequences (lists, strings, ranges) |
| **while Loop** | Repeat code while condition is True |
| **break** | Exit loop immediately |
| **continue** | Skip to next iteration |
| **Nested Loops** | Loop inside another loop |
| **loop-else** | Execute block after loop completes normally |

## ðŸ› Troubleshooting

### Common Issues:

**Issue**: "python command not found"
- **Solution**: Python may not be in your PATH. Try `python3` instead

**Issue**: "SyntaxError: unexpected indent"
- **Solution**: Check indentation consistency. Python is whitespace-sensitive

**Issue**: "NameError: name 'x' is not defined"
- **Solution**: Ensure variable is defined before use

**Issue**: "ModuleNotFoundError"
- **Solution**: Check file name and ensure you're in the correct directory
